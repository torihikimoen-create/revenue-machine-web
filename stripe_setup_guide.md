# Stripeセットアップ・マニュアル：自動収益受け取りの第一歩

自動化システムが稼いだ利益を、あなたの銀行口座へ届けるための設定ガイドです。

## 1. 事前準備（用意するもの）
- **メールアドレス**: ビジネス用または普段お使いのもの。
- **銀行口座情報**: 売上の振込先（国内のほぼ全ての銀行が可能です）。
- **本人確認書類**: 運転免許証やパスポート（スマホで撮影してアップロードします）。
- **ウェブサイトURL**: (もし既にあれば) なければ簡単な紹介ページ（Notion等）でも審査は通ります。

## 2. アカウント開設ステップ
1. [Stripe公式サイト](https://stripe.com/jp) へアクセスし、「今すぐ始める」をクリック。
2. 名前、メールアドレス、パスワードを入力してアカウント作成。
3. 届いたメールでアドレス認証を行う。

## 3. ビジネス情報の有効化（重要）
「本番環境を利用する」ために、必要情報を入力します。
1. **事業形態**: 個人事業主（または法人）を選択。
2. **申請内容**: 住所、氏名、事業内容（例：「ITコンサルティング」「事務代行サービス」など）を記入。
3. **入金（振込先）設定**: 銀行の店番、口座番号を入力。

## 4. APIキーの取得（システム連携に必要）
システムが自動で集金するために、以下の2つの秘密の暗号を取得します。
1. Stripeダッシュボードの左メニュー「開発者」をクリック。
2. 「APIキー」タブを選択。
3. **公開可能キー (Publishable key)** と **シークレットキー (Secret key)** を確認します。
   - ※シークレットキーは一度しか表示されないので、大切に保管（あるいは私に教えてください）。

## 5. 安全のための「テストモード」
Stripeには「テストモード」という砂場があります。
- 右上の「テストデータ」のスイッチをONにすると、架空のカードを使って決済のデモができます。
- 最初はこのモードで、システムが正しく「3万円の請求」を出すか一緒に確認しましょう。
