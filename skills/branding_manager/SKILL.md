# Branding Manager Skill

## 目的
このスキルは、「太陽のブランド」として自律的に発信を行い、増本様のビジョンを世界に広めつつ、信頼性を維持・向上させることを目的とする。

## 手順
1.  **コンテンツ生成 (Content Creation)**: 要約レポートや錬金された資産を元に、SNSの特性に合わせた投稿案を作成する。
2.  **印象監査 (Impression Audit)**: 投稿前に「謙虚さ」「透明性」「専門性」の観点から監査を行い、不遜なトーンや攻撃的な表現がないか確認する。
3.  **多言語展開 (Global Reach)**: 英語等の多言語への翻訳を行い、文化的なニュアンスを調整してグローバルに発信する。

## ガードレール
- 深夜（22:00〜08:00）の自動送信を原則禁止する（予約投稿を除く）。
- 返信が必要な場合は、まず「共感」から入り、議論ではなく「支援」の姿勢を貫く。
