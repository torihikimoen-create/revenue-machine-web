# データ整理・命名ルール統一システム (Phase 4)

## 概要
企業のストレージ内に散らばる「名称未設定.pdf」「コピー(1).xlsx」といった乱雑なファイルを、AIが内容を読み取って自動的にリネーム・整理する究極のクリーンアップエンジンです。

## ターゲット収益
- 1社プロジェクト：150,000円〜

## 主要機能
1. **File Scanner**: 階層構造内のファイルをすべてリストアップし、現況を分析。
2. **AI Naming Engine**: ファイルの中身（先頭数行やメタデータ）から、最適なファイル名を「命名規則」に従って生成。
3. **Deduplication**: 内容が同一のファイルを検知し、整理または削除を提案。
4. **Dry-Run & Commit**: 実際にリネームする前に確認用レポートを出し計、承認後に一括適用。

## 価値提供
「必要なファイルが見つからない」という組織の隠れたコストをゼロにします。1社15万円の価値は、全社員の年間検索時間削減コストに換算すれば格安です。
