# SNS Sales Hunter (Phase 4: Sales Automation)

## 概要
SNS（X, Instagram）のプロフィールや投稿をスキャンし、「DMが溜まって困っている」「返信ができない」と発信しているインフルエンサーや企業を特定し、DM整理自動化システムの導入を提案する営業エンジンです。

## 主要機能
1. **Profile Scanner**: 検索キーワード（例：「DM返信遅れます」「DM整理中」）に基づきターゲットを抽出。
2. **Context Analyzer**: 相手の投稿スタイルやトーンを分析し、最適な提案のタイミングを計る。
3. **DM Draft Generator**: 相手のトーンに合わせた、不快感を与えない自然な提案文（DM）を作成。
4. **CRM Sync**: 提案状況を管理し、継続的なフォローアップを支援。

## 価値
営業マンが1日中SNSを見張る代わりに、AIが24時間体制で「困っている人」を世界中から探し出し、チャンスを逃さずアプローチします。
