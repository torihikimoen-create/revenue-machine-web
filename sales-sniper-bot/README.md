# Google Maps Lead Sniper (Phase 4: Sales Automation)

## 概要
Google Maps Platform APIを使用して、特定の地域・業種の店舗情報をスキャンし、「口コミへの返信が滞っている」店舗を自動的にリストアップする営業支援エンジンです。

## 主要機能
1. **Place Searcher**: 特定のキーワード（例：レストラン、美容室）と地域で店舗を検索。
2. **Review Analyzer**: 直近の口コミを取得し、オーナーからの返信有無をチェック。
3. **Lead Score**: 「未返信数」「平均評価」「最終更新日」から営業優先順位を算出。
4. **CRM Export**: 抽出したリード情報をCSV形式で出力。

## 営業への活用
抽出された「要改善」店舗に対し、自動で生成されたパーソナライズ提案（例：「○件の口コミが未返信のままです。AIで解決しませんか？」）を送付する材料となります。
