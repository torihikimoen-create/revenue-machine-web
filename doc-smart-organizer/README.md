# AI書類リサーチ・事前整理システム (Phase 3)

## 概要
契約書、領収書、レポートなどの大量の書類から、必要な情報（日付、金額、当事者、核心内容）をAIが自動で抽出し、整理・要約するシステムです。

## ターゲット収益
- 1案件：50,000円〜

## 主要機能
1. **Multi-Format Loader**: PDF, Word, 画像書類のテキスト化（OCR連携）。
2. **Metadata Extractor**: 日付、金額、取引先等の構造化データの抽出。
3. **Smart Summarizer**: 書類の内容を3行で要約し、重要度を判定。
4. **Report Generator**: 整理結果をCSVまたはリサーチレポート形式で出力。

## 価値提供
人間が数時間かけて行う「書類の読み込みとエクセル転記」を数秒で完了させます。1件5万円の単価は、専門職（事務・法務補助）の人件費削減効果に基づいています。
