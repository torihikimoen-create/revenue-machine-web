# Stripe テストモード設定・かんたん手順書

審査を待たずに、今すぐ「10日間トライアル」の動きを試すための手順です。

## ステップ 1：テストモードに切り替える
Stripe ダッシュボードの右上にある **「テストモード」** というスイッチをオンにしてください。
（画面全体がオレンジ色の帯などで「テストモード」と表示されれば成功です）

## ステップ 2：テスト用商品を作る
1. 左メニューの **「商品カタログ」** をクリック。
2. **「商品を追加」** をクリック。
3. 以下を入力します：
    - **名前**: `AETHER CORE - 10-Day Trial Test`
    - **価格**: `33000` 円
    - **継続区分**: `毎月`
4. **「追加オプション」** または **「無料トライアル」** という項目を探し、 **`10` 日間** に設定します。
5. 右上の **「商品を保存」** をクリック。

## ステップ 3：支払いリンク（Payment Link）を作る
1. 保存した商品の詳細画面にある **「支払いリンクを作成」** ボタンを押します。
2. 設定はそのままで **「リンクを作成」** を押します。
3. `https://buy.stripe.com/test_...` というリンクが生成されるので、それをコピーして私に教えてください！

## ステップ 4：動作確認（私がやります）
リンクを教えていただければ、私が以下のテストカード番号を使って、「0円で決済が完了し、10日後に課金が始まる予約が入るか」をチェックします。

*   **テスト用カード番号**: `4242 4242 4242 4242`
*   **有効期限**: `12/31` (未来なら何でも可)
*   **CVC**: `123`

---

このステップ 3 までのリンク教えていただければ、すぐにサイトへ反映します！操作中に迷ったら、その画面のスクリーンショットをまた見せてくださいね。
