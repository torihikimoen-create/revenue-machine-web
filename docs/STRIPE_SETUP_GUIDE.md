# Stripe サブスクリプション & 顧客ポータル設定ガイド

ウェブサイト側の準備は整いました。最後に、Stripe ダッシュボードで以下の設定を行うことで、10日間のトライアルと自動解約システムが稼働します。

## 1. 商品（サブスクリプションプラン）の作成
10日間の無料トライアルを含む商品を作成します。

1. [Stripe ダッシュボード](https://dashboard.stripe.com/test/products)（まずはテストモード推奨）を開きます。
2. 「商品を追加」ボタンを押します。
    - **商品名**: AETHER CORE - Professional Plan (または任意)
    - **価格モデル**: 継続
    - **料金**: 33,000円 (例) / 毎月
3. 「無料トライアル」にチェックを入れ、**「10日間」** を入力します。
4. 「保存」して、生成された **「支払いリンク（Payment Link）」** をコピーしてください。
    - コピーしたリンクを `index.html` の `Start 10-Day Free Trial` ボタンの `href` に貼り付けます。

## 2. 顧客ポータル（セルフ解約）の有効化
顧客が自分で解約できるように設定します。

1. [設定 > 顧客ポータル](https://dashboard.stripe.com/test/settings/billing/portal) を開きます。
2. **「サブスクリプションのキャンセルを許可する」** をオンにします。
    - **キャンセルのタイミング**: 「現在の請求期間の終了時にキャンセルする」を選択（推奨）。
    - **解約理由**: 「キャンセル理由を顧客に尋ねる」をオン。
3. **「ビジネス情報を表示する」** をオンにし、特商法ページへのリンク（`tokushoho.html`）をセットします。
4. 設定を保存すると生成される **「ポータルリンク」** をコピーしてください。
    - これを `index.html` フッターの `portal-link` クラスがついた `<a>` タグの `href` に貼り付けます。

---

## 3. 自動通知の設定（リマインド）
トライアル終了前に自動でメールが飛ぶようにします。

1. [設定 > 請求 > 顧客へのメール](https://dashboard.stripe.com/test/settings/billing/emails) を開きます。
2. **「トライアル終了のリマインドメール」** をオンにします。
    - これにより、トライアル終了の3日前に顧客へ「間もなく課金が始まります」という通知が自動で届き、トラブルを防止できます。

---
